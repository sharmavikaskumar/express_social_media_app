 Mini Social Media Web App

small full-stack social media style web application built to practice backend development with Node.js and Express. The project focuses on implementing authentication, database interactions, and common social media features.

## 🚀 Features
- User authentication with JWT
- Password encryption using bcrypt
- Create and view posts
- Like / Unlike posts
- Real-time user search from the database
- Profile page displaying user posts
- Navbar with profile dropdown and logout

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Tailwind CSS

## 📚 Concepts Practiced
- MVC architecture in Express
- RESTful routing
- Middleware for authentication
- JWT-based authentication
- Password hashing with bcrypt
- Database schema design using Mongoose
